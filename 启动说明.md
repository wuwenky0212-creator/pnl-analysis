# 金融市场业务管理系统5.0 - 启动说明

## 快速启动

### 方法1：使用启动脚本（推荐）

1. **启动后端服务**
   - 双击 `start_backend.bat`
   - 等待看到 "Application startup complete" 消息
   - 后端服务运行在：http://localhost:8002

2. **启动前端服务**
   - 双击 `start_frontend.bat`
   - 前端服务运行在：http://localhost:8083

3. **访问系统**
   - 在浏览器中打开：http://localhost:8083

### 方法2：手动启动

#### 启动后端服务
```bash
cd backend
python main.py
```

#### 启动前端服务
```bash
cd frontend
python -m http.server 8083
```

## 常见问题

### 1. 端口被占用
如果提示端口被占用，可以：
- 关闭占用端口的程序
- 或修改 `backend/config.py` 和 `start_frontend.bat` 中的端口号

### 2. 无法访问
- 检查防火墙是否阻止了端口访问
- 确保两个服务都已启动
- 尝试使用 `http://127.0.0.1:8083` 访问

### 3. 其他人无法访问
- 确保他们在同一局域网内
- 使用正确的IP地址（查看"访问链接.html"）
- 检查Windows防火墙设置

## 查看访问链接

打开 `访问链接.html` 文件，可以查看和复制访问链接。
