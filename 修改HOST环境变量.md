# 🔧 修改HOST环境变量

## ❌ 当前问题

HOST的值是 `[::]`，这是IPv6地址格式，但我们的应用需要监听所有接口。

## ✅ 正确的值

HOST应该设置为：`0.0.0.0`

---

## 📝 修改步骤

### 方法1：点击编辑按钮

1. **找到HOST变量**（在变量列表中）
2. **点击HOST变量右侧的编辑图标**（通常是铅笔图标 ✏️）
3. **将值改为**：`0.0.0.0`
4. **点击保存**或按Enter

### 方法2：删除并重新添加

1. **点击HOST变量右侧的删除图标**（垃圾桶图标 🗑️）
2. **点击 "+ New Variable"** 按钮
3. **变量名**：`HOST`
4. **变量值**：`0.0.0.0`
5. **点击保存**

---

## ✅ 检查所有环境变量

确保以下变量都已正确配置：

| 变量名 | 值 | 状态 |
|--------|-----|------|
| `HOST` | `0.0.0.0` | ⚠️ 需要修改 |
| `PORT` | `8002` | ✅ 应该正确 |
| `DEBUG` | `False` | ✅ 已配置 |
| `DATA_FORMAT` | `mock` | ✅ 已配置 |
| `DEFAULT_CURRENCY` | `CNY` | ✅ 已配置 |
| `DEFAULT_UNIT` | `万元` | ✅ 已配置 |

---

## 🚀 修改后

修改完成后：
1. Railway会自动保存更改
2. 可能需要点击 **"Deploy"** 按钮重新部署
3. 或者Railway会自动触发重新部署

---

**现在修改HOST的值为 `0.0.0.0`，然后点击"Deploy"按钮开始部署！** 🚂
