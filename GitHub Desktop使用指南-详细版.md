# 📖 GitHub Desktop 使用指南 - 详细版

## 🎯 添加本地仓库的方法

### 方法1：通过菜单栏

1. **查看窗口顶部**，找到菜单栏：
   - **File**（文件）
   - **Edit**（编辑）
   - **View**（视图）
   - **Repository**（仓库）
   - **Branch**（分支）
   - **Help**（帮助）

2. 如果看不到菜单栏：
   - 按 **`Alt`** 键显示菜单栏
   - 或者点击窗口左上角

3. **点击 "File"** → **"Add Local Repository"**

---

### 方法2：通过欢迎界面

如果是第一次打开GitHub Desktop，会看到欢迎界面：

1. **点击 "Add"** 按钮（在界面中央）
2. 选择 **"Add Existing Repository from Hard Drive"**
3. 点击 **"Choose..."**
4. 选择项目文件夹：`D:\workspace\损益归因分析`
5. 点击 **"Add Repository"**

---

### 方法3：通过Repository菜单

1. 点击顶部菜单栏的 **"Repository"**
2. 选择 **"Add"** → **"Add Existing Repository"**
3. 点击 **"Choose..."**
4. 选择项目文件夹：`D:\workspace\损益归因分析`
5. 点击 **"Add Repository"**

---

### 方法4：直接拖拽

1. 打开文件管理器，找到项目文件夹：`D:\workspace\损益归因分析`
2. **直接拖拽**文件夹到GitHub Desktop窗口
3. GitHub Desktop会自动识别并添加

## 📋 完整的操作流程（参考）

### 步骤1：打开GitHub Desktop
- 如果还没打开，双击桌面上的GitHub Desktop图标

### 步骤2：添加本地仓库
- 使用上面的任意一种方法

### 步骤3：选择项目文件夹
- 路径：`D:\workspace\损益归因分析`
- 点击文件夹后，点击 **"Select Folder"** 或 **"确定"**

### 步骤4：提交代码
- 在GitHub Desktop左侧，你会看到所有文件
- 左下角输入提交信息：`初始提交 - 损益归因分析系统`
- 点击 **"Commit to main"**

### 步骤5：推送代码
- 点击右上角的 **"Publish repository"** 按钮
- 确认仓库名称：`pnl-analysis`
- 点击 **"Publish Repository"**
