# 🚀 执行步骤说明

## 📋 第一步：在GitHub创建仓库（必须）

**在浏览器中操作**：

1. 访问：https://github.com/new
2. 填写信息：
   - Repository name: `pnl-analysis`
   - 选择：**Public**
   - 不要勾选任何选项
3. 点击 "Create repository"

**重要**：必须先创建仓库，否则推送会失败！

---

## 🚀 第二步：执行批处理脚本

### 方法1：双击运行（最简单）

1. 在项目文件夹中找到：`推送到GitHub.bat`
2. **双击运行**
3. 按照提示操作

### 方法2：在PowerShell中运行

1. 在项目文件夹中，按住 **Shift**，右键点击空白处
2. 选择 "在此处打开PowerShell窗口"
3. 执行：
   ```powershell
   .\推送到GitHub.bat
   ```

---

## ⚠️ 重要提示

### 如果提示需要登录GitHub：

1. **用户名**：输入 `wuwenky0212-creator`
2. **密码**：**不是你的GitHub密码**！

### 需要生成Personal Access Token：

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. Note: `Railway部署`
4. 勾选权限：**`repo`**（全部勾选）
5. 点击 "Generate token"
6. **复制token**（只显示一次！）
7. 在脚本提示输入密码时，粘贴这个token

---

## ✅ 推送成功后

代码推送成功后，下一步：

1. 访问：https://railway.app
2. 使用GitHub账号登录
3. 点击 "New Project"
4. 选择 "Deploy from GitHub repo"
5. 选择仓库：`wuwenky0212-creator/pnl-analysis`

---
